# tree-sitter-zig-spm

Swift Package Manager wrapper for the [tree-sitter-zig](https://github.com/tree-sitter-grammars/tree-sitter-zig) grammar.

## Usage

Add this package to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/burin-labs/tree-sitter-zig-spm.git", from: "0.1.0")
]
```

Then add `TreeSitterZig` as a dependency of your target.
